## Infraestrutura Técnica

### Cyber Range Cyberbit

**Descrição:** Plataforma enterprise de simulação de SOC com cenários realistas.

**Capacidades:**
- Simulação de ataques em tempo real
- Ambientes de rede virtualizados
- Cenários de Red Team / Blue Team
- Exercícios de resposta a incidentes
- Métricas e relatórios de desempenho

**Cenários Disponíveis:**
1. APT Simulation
2. Ransomware Attack
3. Insider Threat
4. DDoS Attack
5. Supply Chain Attack

### Cloud Labs

**AWS:**
- Amazon SageMaker (ML)
- Amazon Bedrock (LLMs)
- AWS Security Hub
- Amazon GuardDuty

**Azure:**
- Azure Machine Learning
- Azure OpenAI Service
- Microsoft Sentinel
- Azure Defender

**GCP:**
- Vertex AI
- Chronicle Security
- Security Command Center

### Jupyter Hub

**Configuração:**
- JupyterHub multi-user
- GPU support (NVIDIA T4/A10)
- Pre-installed libraries
- Persistent storage
- Git integration

**Libraries Pré-instaladas:**
```
numpy, pandas, scikit-learn
tensorflow, pytorch, keras
transformers, langchain
openai, anthropic
scapy, pefile
matplotlib, seaborn, plotly
```

- Integração com AVA

**Categorias de Challenges:**
- Crypto

- Misc

## Políticas de Laboratório

### Uso Ético
- Práticas apenas em ambientes autorizados
- Proibido uso de conhecimentos para fins maliciosos
- Confidencialidade de vulnerabilidades encontradas

### Compartilhamento
- Proibido compartilhar credenciais
- Proibido divulgar soluções de CTF
- Colaboração incentivada em atividades de grupo

### Recursos
- Fair use de recursos computacionais
- Não minerar criptomoedas
- Reportar uso indevido
